WHAT ARE MACROS?

Macros are code-based commands used to automate PowerPoint processes.

Macros are disabled by default, but for many of these PowerPoints, they must be enabled in order for the PowerPoint to function. They are the basis for scoreboards, guided set ups, and other functionalities.

If you are required to enable macros on one of our PowerPoints, you can find instructions on how to do so within the PowerPoints themselves or get the most up to date information on the Microsoft Help website. Microsoft automatically disables Macros because they can sometimes contain malicious code. I do not put any malicious code in my PowerPoints, but you should always be careful when downloading anything from the internet.

.PPTM VS .POTM

.PPTM and .POTM are two types of PowerPoints. Both contain macros, but are nevertheless distinct.

.PPTM files behaves like any normal PowerPoint, just with additional macros that must be enabled upon opening the file. POTM is what PowerPoint calls a "template file," where if you open the file, it automatically creates a duplicate that must then be saved as a separate PPTM file. POTMs are useful if you only want to download a template once and create multiple games from it.
